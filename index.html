About

현장 경험을 서비스 설계 감각으로 연결합니다.

유통영업, 생산관리, 고객관리, 재고관리, 인력관리 업무를 오래 경험했습니다. 이후 Java와 Spring Boot 중심의 풀스택 개발을 학습하며, 사용자 화면과 운영자 관리 흐름이 함께 맞물리는 프로젝트를 만들어왔습니다.

관심 분야는 Java/Spring Boot 백엔드, IT 서비스 운영, 기술영업, AI 도구를 활용한 개발 생산성 향상입니다. 기능 자체보다 그 기능이 어떤 문제를 해결하고 어떤 흐름 안에서 쓰이는지를 중요하게 봅니다.

Strengths

제가 프로젝트에서 계속 가져가는 관점

사용자와 운영자의 흐름을 함께 봅니다.

고객, 직원, 재고, 운영 흐름을 다뤄온 경험을 바탕으로 사용자 화면 뒤에서 운영자가 확인해야 할 데이터와 업무까지 함께 생각합니다.

새로운 기술을 서비스 기능으로 연결합니다.

MediaPipe, OCR, Kakao Map API 같은 기술을 단순 적용으로 끝내지 않고, 사용자가 이해할 수 있는 피드백과 화면 흐름으로 연결하려고 합니다.

요구사항을 실행 가능한 단위로 정리합니다.

막연한 아이디어를 역할, 화면, API, 데이터 흐름으로 나누고 우선순위를 정해 하나씩 구현하는 방식으로 프로젝트를 진행합니다.
